A small plugin for Notification style popups on your website

Getting started

To start a notification queue simply initialize the object/class:

const queue = new NotifyQueue();

Standard Notification

Standard notifications are super simple to use, you only need to add a message to the function call!

queue.notify({
        message: 'This is a notification snackbar! 
 You can add HTML elements like links to this element.',
    });
info_outlineTry it out

Custom Notification

Custom notifications are made by changing the style and content objects of a notification. Alternatively, you may use the setDefaultStyle function to override all styles.

queue.notify({
        style: {
            opacity: 0.6,
            iconSize: '50px',
            backgroundColor: 'red',
            fontSize: '24px',
            buttonBackgroundColor: 'blue',
        },
        actionText: 'Thanks!',
        message: 'We saved your items!',
    });
info_outlineTry it out

Duration Notification

By adding a duration value to your function call, the notification will persist for the specified duration!

queue.notify({
        message: 'This will stay on screen for 5 sec before destroying itself',
        duration: 5000,
    });
info_outlineTry it out

Custom Action Notification

The user action event (pressing the button) can be bound when calling the notification.

queue.notify({
        message: 'This snackbar has custom logic when its action button is pressed',
        action: function () {
            alert('Pressed the action button!')
            this.close();
        },
    });
info_outlineTry it out

Notification Queue

NotificationQueue automatically queues notifications that recieve a duration.
